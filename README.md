# base64.js

base64的编码和解码！

Base64编码，是一种基于64个可打印字符（A-Z,a-z,0-9,+/这64个字符）来表示二进制数据的表示方法。

关于该编码的更多内容请参见 [关于Base64编码](http://www.cnblogs.com/liyiwen/p/3814968.html)。

## 使用base64

### base64的编码

```js

base64.encode('1234abcd编码测试');

```

### base64的解码

```js

base64.decode('MTIzNGFiY2TnvJbnoIHmtYvor5U');

```
